# Placeholder Image

This project uses SafeImage component that displays a placeholder when image URIs are null or empty.

The placeholder.png file should be a simple gray/transparent image. 

For now, the SafeImage component will show a gray background when no valid URI is provided.

To add an actual placeholder.png file:
1. Create a simple 1x1 or 100x100 px gray/transparent image
2. Place it at assets/placeholder.png
3. The SafeImage component will automatically use it
